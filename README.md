# Credit Card Fraud Detection

## Description

This project uses Machine Learning techniques to detect fraudulent credit card transactions. The model is trained on transaction data to classify transactions as legitimate or fraudulent based on various features and patterns. The objective is to improve fraud detection accuracy and help financial institutions identify suspicious activities in real time.

## Features

* Data preprocessing and cleaning
* Exploratory Data Analysis (EDA)
* Feature selection and scaling
* Machine Learning model training
* Fraudulent transaction prediction
